⚙ Configurar base de datos

Esta app usa Supabase para guardar los datos en la nube gratis. Sigue los pasos abajo para configurarla una sola vez.

Paso 1: Ve a supabase.com y crea una cuenta gratis.
Paso 2: Crea un proyecto nuevo (escoge cualquier región).
Paso 3: Ve a SQL Editor y pega este código para crear la tabla: 
CREATE TABLE guianzas (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  fecha date NOT NULL,
  tipo text NOT NULL,
  empresa text NOT NULL,
  personas integer,
  guia_nombre text NOT NULL,
  otro_guia text,
  pago_total numeric NOT NULL,
  detalle_calculo text,
  notas text,
  creado_en timestamptz DEFAULT now()
);
Paso 4: Ve a Project Settings → API y copia la Project URL y la anon public key.
Paso 5: Pégalas abajo y guarda.
✓ Configuración guardada. Ahora puedes registrar guianzas.
⚠ Configura Supabase primero

Para guardar los datos en la nube necesitas conectar tu base de datos. Es gratis y tarda 5 minutos.
